linux常用命令

By zxj at: 2018-03-12
linux命令

查看日志
tail -f /home/tomcatapp/logs/app/wallet/wallet_info.log
tail -f /home/tomcatapp/logs/app/wallet/wallet_error.log
tail -f /home/tomcatapp/logs/catalina.out

启动、停止 ./shutdown.sh ./startup.sh
cd /home/tomcatapp/bin/ 进入x目录
ps -ef|grep tomcat 查看xx进程的信息
kill -9 10136 杀死某个进程
netstat -apn|grep 8080 查看进程
lsof -i:8080 查看8080端口占用情况
netstat -ntlp 列出所有端口

ls 显示当前目录的文件和目录
ls -l 显示当前目录的文件和目录的详细信息
ls -a 显示当前目录的文件和目录,包括隐藏文件和目录
ls -al 多个选项合并使用
ll 命令列出的信息更加详细，有时间，是否可读写等信息

cd / 跳转到根目录
cd ~ 跳转到当前用户的家目录
cd - 切换到上次进入的目录
cd abc/test : 使用相对路径的方式找到abc/text目录,并切换到该目录
cd /abc/test: 使用绝对路径(从根目录下面开始) 寻找abc/test目录,并切换到该目录

history 用于显示执行过的历史命令记录
pwd 显示当前所在的目录
clear 清除屏幕信息
df:用于显示磁盘的使用情况
df -h: 格式化显示输出磁盘使用情况
du 目录名称 显示每个文件的大小
du -sh 目录名称 汇总显示该目录以及子目录下所有文件的大小
rm 删除文件
top 查看cpu
echo 输出
free 内存使用

init 0: 关机
init 6: 重启
grep zhuxiaojin index.html 查找字符串‘zhuxiaojin ’在文件中的位置
mv index.html index234234.html 前面是被修改文件名 要修改成怎样的文件名
cat /etc/sysconfig/network-scripts/ifcfg-eth0 ip设置,dns设置
ifconfig eth0 192.168.121.22 设置ip地址
service network restart 重启网络服务

vi编辑：
vi /path/to/file 打开文件
vi 编辑模式:
shift+G,跳到最后一行
shift+4,跳到行尾
数字0,跳到行首
gg,跳到第一行
ctrl+f 向下翻一页
ctrl+b 向上翻一页
3dd:从光标行开始删除3行
dd 删除一行
2yy 从光标行开始复制2行
o或者i进入插入模式:
yy 复制
p 粘贴
esc退出插入模式
shift+: 末行模式:
wq: 保存退出
q!: 退出不保存
1,3d 删除1-3行
set nu,显示行号
num: 跳转到某一行
1,$s/zhuxiaojing/zhuge/gi 替换字符串

创建文件
touch 文件名
vi 文件名
创建目录
mkdir 目录名
mkdir -p 目录名1/目录名2 创建层级目录 可以是相对目录,也可以是绝对目录
删除文件或目录
rm 文件名
rm -f 文件名
rm -fr 目录 删除目录必须使用 -r 递归删除
查找文件
cat
less/more
tail
grep
find 例子:find /etc -name ‘srm*’ find / -name httpd.conf
打包和压缩
tar -cvf 例子:tar -cvf test.jar test
tar -zcvf 例子:tar -cvf test.jar.gz test
tar -xvf 例子:tar -xvf test.tar
tar -zxvf 例子:tar -xvf test.tar.gz

常用配置文件
1 /etc/profile 环境变量配置文件
2 /etc/sysconfig/iptables 防火墙的相关配置
3 /etc/sysconfig/network-scripts/ifcfg-eth0 网卡配置文件
4 /etc/inittab 系统运行级别配置文件
5 /etc/rc.d/rc.local 配置系统开机启动相关命令
6 /etc/fstab 文件系统配置文件
